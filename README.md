# Task 4: SQL Data Analysis

## 📌 Objective
The objective of this project is to perform data analysis using SQL queries on an Ecommerce database. It demonstrates how to retrieve, filter, join, summarize, and optimize structured data using MySQL.

## 🛠️ Tools Used
- MySQL Workbench
- SQL
- GitHub

## 📂 Dataset
Custom Ecommerce SQL Database

## 📚 SQL Concepts Covered
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- Aggregate Functions (SUM, AVG)
- Subqueries
- Views
- Indexes
